# tokio-tutorial-pdf

Generates a PDF version of the [tokio-rs tutorial](https://github.com/tokio-rs/website/tree/master/content/tokio)

# How-to use

```bash
$ nix build github:abhillman/tokio-tutorial-pdf --no-write-lock-file
$ ls result
tokio-tutorial.pdf
```
